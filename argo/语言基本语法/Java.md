### 数据类型
#### 基本数据类型和引用数据类型
- 基本数据类型
	- 字节型
		- byte 1个字节
	- 短整型
		- short 2个字节
	- 整型
		- int 4个字节
	- 长整型
		- long 8个字节
	- 字符型
		- char 2个字节
	- 单精度浮点型
		- float 4个字节
	- 双精度浮点型
		- double 8个字节
	- 布尔型
		- boolean 1个字节
- 引用数据类型
	- 类
		- class
	- 接口
		- interface
	- 数组
		- [ ]
